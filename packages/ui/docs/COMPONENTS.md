# 组件列表 (Components)

这里是 `pd-ui` 提供的所有核心组件的详细文档。

## 基础组件 (General)

- [Button 按钮](./components/button.md) - 用于触发操作的基础按钮。

## 表单组件 (Form)

- [Input 输入框](./components/input.md) - 基础文本输入。
- [Label 标签](./components/label.md) - 配合表单控件使用的标签。
- [Checkbox 复选框](./components/checkbox.md) - 用于多项选择。
- [Switch 开关](./components/switch.md) - 状态切换。
- [Select 选择器](./components/select.md) - 下拉选择列表。

## 展示组件 (Data Display)

- [Badge 徽章](./components/badge.md) - 标记状态或数量。
- [Avatar 头像](./components/avatar.md) - 展示用户头像。
- [Accordion 手风琴](./components/accordion.md) - 折叠展示内容。
- [Tabs 标签页](./components/tabs.md) - 内容分栏展示。
- [Table 表格](./components/table.md) - 结构化数据展示。

## 反馈与交互 (Feedback & Overlay)

- [Dialog 对话框](./components/dialog.md) - 模态弹窗。
- [Sheet 侧边栏](./components/sheet.md) - 从侧边滑出的面板。
- [Progress 进度条](./components/progress.md) - 任务进度展示。
- [Skeleton 骨架屏](./components/skeleton.md) - 加载占位展示。

> 提示：如果某个组件文档尚未完善，你可以直接查阅 [Storybook](http://localhost:6006) 中的示例代码。
