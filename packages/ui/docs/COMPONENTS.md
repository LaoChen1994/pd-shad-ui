# 组件列表 (Components)

这里是 `pd-shad-ui` 提供的所有核心组件的详细文档。

## 基础组件 (General)

- [Button 按钮](./components/button.md) - 用于触发操作的基础按钮。
- [Badge 徽章](./components/badge.md) - 标记状态或数量。
- [Separator 分隔符](./components/separator.md) - 视觉分割线。
- [Toggle 开关按钮](./components/toggle.md) - 切换状态的按钮。
- [Toggle Group 开关按钮组](./components/toggle-group.md) - 多个开关状态的组合。

## 表单组件 (Form)

- [Input 输入框](./components/input.md) - 基础文本输入。
- [Label 标签](./components/label.md) - 配合表单控件使用的标签。
- [Checkbox 复选框](./components/checkbox.md) - 用于多项选择。
- [Switch 开关](./components/switch.md) - 状态切换。
- [Select 选择器](./components/select.md) - 下拉选择列表。
- [Textarea 多行文本](./components/textarea.md) - 输入长文本。
- [Slider 滑块](./components/slider.md) - 在范围中滑动选择。
- [Input OTP 验证码](./components/input-otp.md) - 验证码输入框。
- [Calendar 日历](./components/calendar.md) - 日期选择组件。
- [Form 表单](./components/form.md) - 集成了校验功能的表单容器。

## 展示组件 (Data Display)

- [Avatar 头像](./components/avatar.md) - 展示用户头像。
- [Accordion 手风琴](./components/accordion.md) - 折叠展示内容。
- [Tabs 标签页](./components/tabs.md) - 内容分栏展示。
- [Table 表格](./components/table.md) - 结构化数据展示。
- [Card 卡片](./components/card.md) - 承载业务信息的容器。
- [Carousel 轮播图](./components/carousel.md) - 图片或内容的轮播展示。
- [Aspect Ratio 宽高比](./components/aspect-ratio.md) - 固定内容的展示比例。

## 导航组件 (Navigation)

- [Breadcrumb 面包屑](./components/breadcrumb.md) - 层级路径展示。
- [Navigation Menu 导航菜单](./components/navigation-menu.md) - 复杂的顶部导航。
- [Pagination 分页](./components/pagination.md) - 内容的分页导航。
- [Menubar 菜单栏](./components/menubar.md) - 顶部的窗口级功能菜单。

## 反馈与交互 (Feedback & Overlay)

- [Dialog 对话框](./components/dialog.md) - 模态弹窗。
- [AlertDialog 警示对话框](./components/alert-dialog.md) - 需要强确认的警示窗。
- [Sheet 侧边栏](./components/sheet.md) - 从侧边滑出的面板。
- [Popover 气泡卡片](./components/popover.md) - 触发显示的轻量内容。
- [Tooltip 文字提示](./components/tooltip.md) - 悬停展示的极简说明。
- [Hover Card 悬停卡片](./components/hover-card.md) - 展示链接预览的浮动窗。
- [ContextMenu 右键菜单](./components/context-menu.md) - 鼠标右键触发的功能菜单。
- [DropdownMenu 下拉菜单](./components/dropdown-menu.md) - 点击触发的功能列表。
- [Sonner 提示](./components/sonner.md) - 全局 Toast 提示。
- [Progress 进度条](./components/progress.md) - 任务进度展示。
- [Skeleton 骨架屏](./components/skeleton.md) - 加载占位展示。

## 布局组件 (Layout)

- [Scroll Area 滚动区域](./components/scroll-area.md) - 自定义样式的滚动容器。
- [Resizable 可调布局](./components/resizable.md) - 支持用户拖拽调整大小的布局。

> 提示：如果某个组件文档尚未完善，你可以直接查阅 [Storybook](http://localhost:6006) 中的示例代码。
